# privateGP

## Project Overview

`privateGP` is a work-in-progress project exploring ways to build private and secure generative or predictive models using Gaussian Processes. The goal is to provide techniques and experiments that help protect sensitive data while enabling machine learning workflows.

## Installation

There is no packaged release yet. To experiment with the code:

```bash
# Clone the repository
git clone <repo-url>
cd privateGP
```

More detailed setup instructions will be added once the project structure stabilizes.
