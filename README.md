# privateGP

## Project Goals
- Enable users to run privacy-focused generative AI models locally.
- Provide a minimal interface for interacting with models without relying on external services.
- Encourage experimentation with self-hosted AI.

## Features
- Lightweight setup for running models on personal hardware.
- Simple command-line and web interfaces.
- Open-source and community driven.

## Current Status
This project is in its early stages. The repository currently contains initial planning files and basic instructions. Core features are under active development.

## Prerequisites
- **Python 3.8+** is recommended.
- Git for cloning the repository.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/privateGP.git
   ```
2. Change into the project directory:
   ```bash
   cd privateGP
   ```
3. (Optional) Create and activate a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
