# privateGP

privateGP is a telehealth platform designed to streamline remote consultations between general practitioners and patients. The project aims to offer secure communication and integrate common Australian healthcare requirements.

## Planned Features
- AI-assisted triage and note generation
- Links to PBS (Pharmaceutical Benefits Scheme) and MBS (Medicare Benefits Schedule) resources
- End-to-end encryption and multi-factor authentication
- Secure document storage and retrieval
- Video consultation capabilities

## Setup
1. Install Python 3.11 or later.
2. Clone this repository.
3. Install dependencies with `pip install -r requirements.txt` (file to be provided).
4. Run any required database migrations.
5. Start the application using `python app.py`.
