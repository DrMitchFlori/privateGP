# privateGP

## Goal
The privateGP application is a telehealth platform designed to connect patients with health professionals securely. It focuses on safeguarding patient privacy while enabling convenient remote consultations and medical record management.

## Prerequisites
- Node.js 18+ or Python 3.10+
- Git
- Docker (optional, for containerized development)

## Running the Application
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd privateGP
   ```
2. Install dependencies:
   ```bash
   npm install  # or pip install -r requirements.txt
   ```
3. Start the development server:
   ```bash
   npm start  # or appropriate command
   ```
